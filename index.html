User Management API

made by salam adil | Express js / Postgresql

Welcome to the User Management API documentation. This API provides endpoints for managing user information efficiently. Below is an overview of the available endpoints and their functionalities.

Usage

Endpoints

Endpoint Method Description
/api/users GET Get a list of all users.
/api/users POST Create a new user.
/api/users/:id PUT Update user details by ID.
/api/users/:id DELETE Delete a user by ID.
/api/users/search GET Search users by name or username.
/api/users/filter GET Filter users by city.
/api/users/first GET Get the first user.
/api/users/last GET Get the last user.
/api/users/:id GET Get auser by ID.
/api/users/:id/geo GET Get a user's geo information and Google Maps link by ID.
