User and Employee Management API

Made by Salam Adil | Express.js / PostgreSQL

Welcome to the User and Employee Management API documentation. This API provides endpoints for managing user and employee information efficiently. Below is an overview of the available endpoints and their functionalities.

Usage

Endpoints for Users

Endpoint Method Description
/api/users GET Get a list of all users.
/api/users POST Create a new user.
/api/users/:id PUT Update user details by ID.
/api/users/:id DELETE Delete a user by ID.
/api/users/search GET Search users by name or username.
/api/users/filter GET Filter users by city.
/api/users/first GET Get the first user.
/api/users/last GET Get the last user.
/api/users/:id GET Get a user by ID.
/api/users/:id/geo GET Get a user's geo information and Google Maps link by ID.

Endpoints for Employees

Endpoint Method Description
/api/employees GET Get a list of all employees.
/api/employees POST Create a new employee.
/api/employees/:id PUT Update employee details by ID.
/api/employees/:id DELETE Delete an employee by ID.
/api/employees/search GET Search employees by name or email.
/api/employees/filter GET Filter employees by department.
/api/employees/first GET Get the first employee.
/api/employees/last GET Get the last employee.
/api/employees/:id GET Get an employee by ID.
